# Tooling

# Prerequisites

A macOS operating system is required to fully follow the training. However, you can use Windows or Linux if you're only interesed in learning the Swift langugage

# Xcode

Install the latest Xcode version from the AppStore. You need to run Xcode at least once before starting the training to perform a post setup steps. At the end of the this phase, you should see a welcome window.

# Using a free or paid Apple developr account

If you want to deploy your app to a device, you need to have at least a free developer account. A paid account, will allow to develop and test on multiple devices and, most importantly,

If you are part of a company, you can request to be added to your company account as a developer. Once this is done, you'll need to add some certificates to your keychain.

If you want to use a self-owned account You can create one https://developer.apple.com/open in new window.

Since a free account allows to deploy to a single device at a time, this should be enough for this training. Otherwise, you can use any paid account that is available to you.

Once you have a developer account (as part of your enterprise or self-owned), you need to add it to Xcode. This is done by opneing Xcode and then choosing "Xcode -> Settings -> accounts tab" and then click on the "+" button. Once added, the account should be visible in the accounts list.

# If you want to only learn Swift

If you don't have access to a macOS system or want to just learn the swift language, you can install the standalone swift toolchain from this linkopen in new window. Please follow the installation instructions that correspond to your system: windowsopen in new window, linuxopen in new window and macOSopen in new window.
